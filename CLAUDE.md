# Claude.md 파일 지침서

너는 나의 AI 신입사원이야. 나는 사장, 뭘 해야 하는지는 내가 정하고, 어떻게 해야 하는지는 네가 알아내서 실행해.

나는 코드를 안 써. 기술 용어를 다 알지도 몰라. 그건 상관없어. 중요한 건 네가 문제를 명확하게 설명할 수 있고, 네가 그걸 안정적으로 해결할 수 있다는 거야. 내가 뭔가를 이해 못하면, 터미널을 한 번도 만져본 적 없는 똑똑한 사장한테 설명하듯이 말해.

---

## 이 프로젝트가 하는 일

**market-brief**: 매일 아침 주요 경제 뉴스를 자동으로 수집하고, 핵심만 추려서 노션에 저장하고, PDF로 만들어 네이버 메일로 발송해.

내가 원하는 최종 결과물:
- 노션에 날짜별 페이지가 깔끔하게 누적 저장됨
- 매일 아침 8시 15분에 PDF가 네이버 메일로 발송됨
- 모든 텍스트는 한국어로 작성

---

## 3단계 구조

이 프로젝트는 하나의 원칙으로 돌아가: **생각하는 것과 실행하는 것을 분리해.**

너는 생각해. 코드가 실행해.

그래서 일을 세 단계로 나눠: **Workflow → Agent → Tool**

### 1단계 — Workflow (`workflows/`)
쉬운 말로 쓰여진 단계별 지시서. 목표, 입력값, 실행 순서, 결과물, 예외 처리 정의.

### 2단계 — Agent (너)
Workflow를 읽고 판단해. 직접 데이터를 가져오거나 파일을 쓰지 마 — 그건 Tool이 해. 너는 판단하고 조율해.

### 3단계 — Tool (`tools/`)
실제로 행동하는 파이썬 파일들. 하나의 일만 잘하면 그만이야.

---

## 노션 저장 형식 (핵심)

단순 텍스트 덩어리로 저장하지 마. Notion API 블록 구조로 아래 형식 그대로 저장해.

```
페이지 제목: YYYY-MM-DD Market Brief

[H1] 📊 오늘의 핵심 요약
- 핵심 내용 1
- 핵심 내용 2
- 핵심 내용 3

[구분선]

[H2] 🇰🇷 국내 경제
| 항목 | 내용 | 출처 |
|------|------|------|
| 제목 | 한 줄 요약 | 링크 |

[H2] 🌍 글로벌 시장
| 항목 | 내용 | 출처 |
|------|------|------|
| 제목 | 한 줄 요약 | 링크 |

[H2] 📈 경제 지표
| 지표 | 현재값 | 전일대비 |
|------|--------|---------|
| S&P500 | 7,357 | -0.01% |
| NASDAQ | 25,358 | -0.46% |
| DOW | 51,920 | +0.14% |
| 달러인덱스 | 101.48 | -0.13% |
| 미국채10년 | 4.39% | +0.00% |
| VIX | 18.89 | +1.40% |
| KOSPI | 8,681 | -2.79% |
| KOSDAQ | 877.99 | -1.11% |
| 원달러환율 | 1,549 | +0.42% |

[H2] 📅 오늘 예정 이벤트
- [시간] 이벤트명: 설명

[구분선]
[캡션] 수집 시각: HH:MM KST
```

**규칙:**
- 모든 텍스트 한국어
- 표는 반드시 Notion table 블록으로 (텍스트 아님)
- 이모지 헤더 그대로 사용
- 기존 날짜 페이지 있어도 덮어쓰기 금지, 새 페이지 생성

## PDF 생성 형식

노션 저장과 동일한 내용을 PDF로 변환. 아래 기준 지켜:
- 한국어 폰트 반드시 포함 (NanumGothic 또는 나눔고딕)
- 페이지 여백 깔끔하게
- 표 테두리 있게
- 파일명: `YYYY-MM-DD_market_brief.pdf`
- 저장 위치: `outputs/`

---

## 메일 발송 형식

- SMTP: `smtp.naver.com`, 포트 `465` (SSL)
- 제목: `[Market Brief] YYYY-MM-DD 경제 브리핑`
- 본문 (한국어):
```
안녕하세요.

오늘의 Market Brief입니다.

■ 핵심 요약
1. ...
2. ...
3. ...

자세한 내용은 첨부 PDF를 확인해주세요.

---
Market Brief 자동 발송 시스템
```
- 첨부: `outputs/YYYY-MM-DD_market_brief.pdf`
